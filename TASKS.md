## 🔄 Процесс работы

### 1. **Выбор задачи**
- Проверьте колонку "Статус"
- Убедитесь, что задача не в работе у другого участника
- Назначьте себя ответственным (обновите статус на IN PROGRESS)

### 2. **Работа над задачей**
- Создайте ветку от `main`: `git checkout -b feature/task-id`
- Делайте коммиты с понятными сообщениями
- Обновляйте прогресс в этом файле

### 3. **Завершение задачи**
- Протестируйте свою работу
- Обновите статус на REVIEW
- Создайте Pull Request
- Упомяните других участников для ревью

### 4. **Ревью и мерж**
- Проверьте код коллег
- Оставьте комментарии если нужны правки
- После аппрува - мержите в main
- Обновите статус на DONE
